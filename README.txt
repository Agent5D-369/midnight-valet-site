WAITLIST SETUP
1. Create a form backend account with Formspree or another static-site form service.
2. Create a new form and copy your endpoint, which looks like https://formspree.io/f/your_form_id.
3. Open waitlist.html and replace REPLACE_WITH_YOUR_FORM_ID in the form action.
4. Publish to GitHub Pages. The waitlist form will work on a static site because the submissions go to the external form backend, not GitHub Pages itself.
5. Test a submission. On success, the page will show a link to venue-pilot-scorecard.html.
